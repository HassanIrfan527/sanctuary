# Tmux Keybinds Reference (Prefix: Ctrl+A)

## Tabs (windows)
Ctrl+A + c       → New tab
Ctrl+A + n       → Next tab
Ctrl+A + p       → Previous tab
Ctrl+A + 1-5     → Jump to tab
Ctrl+A + w       → Kill tab

## Splits (panes)
Ctrl+A + v       → Split vertical
Ctrl+A + s       → Split horizontal
Ctrl+A + x       → Kill pane
Ctrl+A + z       → Zoom/unzoom pane

## Pane Navigation (vim-style)
Ctrl+A + h       → Pane left
Ctrl+A + j       → Pane down
Ctrl+A + k       → Pane up
Ctrl+A + l       → Pane right

## Pane Resize
Ctrl+A + H       → Resize left
Ctrl+A + J       → Resize down
Ctrl+A + K       → Resize up
Ctrl+A + L       → Resize right

## Copy Mode
Ctrl+A + [       → Enter copy mode
v                → Begin selection
y                → Copy and exit

## System
Ctrl+A + r       → Reload config
