# My Plan for AGS Rice.

Here are some ideas/logics for this config:

- Minimal, meaning there wouldn't be much bloat or things in the top bar. The top bar would be minimal
- Minimal yet extended functionality. maybe matugen, live wallpaper, custom scripts for my dev workflow (I'll think abt this one though)

### Presets

In my config, preset is a set of themes, color palettes and design layouts:

1. Cozy
2. Pixel (Future plan)
3. Cyberpunk (Future plan)

Note: In version 1, I'm not gonna add more presets which I had planned earlier like the pixel and cyberpunk one. and including a theme switcher. Its a good idea, but for now, I'll keep it simple.

**Idea: Colonists preset** — A rice theme inspired by The Colonists game. Earthy warm palette (soft greens, warm browns, muted orange), workspace names like "Workshop/Farm/Harbor", bar styled as a resource tracker. The cozy "building from nothing" vibe.

### Features

1. Pomodoro timer with pin-functionality to pin/unpin it from the top bar.
2. Live wallpaper. But optional feature, cause it eats system resources.
3. Two sidebars, the right one and the left one:

- Right one is gonna be the notification bar. Including the DND + notification sound functionality.
- Left one is gonna be the productivity sidebar, where I'd put some productivity related things. Like pomodoro timer, tasks or others if wanted.

4. The search modal/popup...would not only find apps, but it'd also be used for calculating numerical problems, or finding files too. I think I can use television for that. That'd be an awesome idea.
5. Rounded screen corners and rounded window borders. **Always**.
6. Transclucent + blurred windows - **The superpower**
7. No Dock for now.
8. The hidden/special workspaces view. When switched, it'd show **Underworld** title on the top and would have special features. I'd have to think abt that though.
9. bucklespring + mechsim + the sound-pack switcher popup - **Must have**
10. Top bar layouts:

- Islands: show only rounded pillboxes/islands
- Full/Maximized: Show a full top bar, but the components of the bar would be separated by a slight rounded pillbox.
- Minimal: Centered, show only a rounded centered bar showing the time, workspaces and the music player. Just simple and minimal.

11. Bottom bar design:
    Lualine-style bottom bar, showing the current music and some other utilities...
